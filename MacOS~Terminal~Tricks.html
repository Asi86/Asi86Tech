Terminal Tricks(Last Updated on 3.27.19.)

Bring Back The Shutdown Menu When Pressing The Power Button
  1. Press the space bar and the command key.
  2. Type in "terminal", then press return once terminal shows up.
  3. Wait 10 seconds.
  4. Type in “defaults write com.apple.loginwindow PowerButtonSleepsSystem -bool no”

To Show Hidden Files, Follow Steps 1-3 And Do The Following Below
  Type “defaults write com.apple.finder AppleShowAllFiles YES”
