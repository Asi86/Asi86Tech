Enable Repeating Keys

  1. Press the command key and the space bar.
  2. Type in "Terminal"
  3. Press return after a black box icon with a <_ is inside it.
  4. Wait 10 seconds.(or until a black window shows up)
  5. Click in the black window (this should enable you to type)
  6. Copy and Paste "defaults write -g ApplePressAndHoldEnabled -bool false" into the window

  1. Press the command key and the space bar.
  2. Type in "Terminal"
  3. Press return after a black box icon with a <_ is inside it.
  4. Wait 10 seconds.(or until a black window shows up)
